네이버 카페 아카이브 매니저 빠른 시작

1. ZIP 압축을 풉니다.
2. NaverCafeArchiveManager 폴더 안의 NaverCafeArchiveManager.exe를 실행합니다.
3. 처음 실행하면 "네이버 로그인 세션 연결" 버튼을 누릅니다.
4. 열린 브라우저에서 네이버에 직접 로그인합니다.
5. 앱으로 돌아와 네이버 카페 게시글 주소 또는 메뉴/게시판 주소를 입력합니다.
6. 다운로드 버튼을 누르면 저장이 시작됩니다.
7. 저장 결과는 NaverCafeArchiveManager 폴더 안의 saved_posts 폴더에 생성됩니다.
8. data 폴더에는 로그인 세션, 저장 목록 인덱스, 메뉴 다운로드 배치 로그가 저장됩니다.

주의:
- data/auth, data/browser_profile에는 개인 로그인 세션 정보가 포함될 수 있습니다.
- 다른 사람에게 프로그램을 공유할 때 기존 data 폴더를 함께 보내지 마세요.
- 다른 PC로 옮긴 뒤에는 그 PC에서 다시 "네이버 로그인 세션 연결"을 실행하세요.
- 문제가 생기면 앱을 종료한 뒤 data 폴더를 삭제하고 다시 로그인 세션을 연결할 수 있습니다.
- 비밀번호는 앱에 저장되지 않습니다.
