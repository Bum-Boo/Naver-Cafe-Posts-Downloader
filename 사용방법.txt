네이버 카페 게시글 저장 도구
==============================

이 도구는 네이버 카페 게시글을 내 컴퓨터에 저장합니다.
게시글 URL을 입력하면 본문 텍스트, HTML, 이미지, 저장 정보가 saved_posts 폴더에 정리됩니다.


1. 준비 사항
------------

- Windows
- Python 3.10 이상 권장
- Chrome 또는 Microsoft Edge
- 네이버 카페 게시글을 볼 수 있는 네이버 계정

먼저 평소 사용하는 Chrome에서 네이버에 로그인해 주세요.


2. 가장 쉬운 실행 방법
---------------------

1. 실행하기.bat 파일을 더블클릭합니다.

2. 처음 사용하는 경우에만 1번 "First install"을 실행합니다.

3. 설치가 끝나면 2번 "Save cafe post"를 실행합니다.

4. 터미널에 저장할 네이버 카페 게시글 URL을 붙여넣고 Enter를 누릅니다.

5. 저장이 완료되면 saved_posts 폴더 아래에 게시글 제목으로 된 폴더가 생깁니다.

다음부터는 1번 설치를 다시 하지 않아도 됩니다.
보통은 실행하기.bat를 열고 2번 "Save cafe post"만 실행하면 됩니다.


3. 저장되는 파일
----------------

게시글마다 아래 파일이 저장됩니다.

- content.txt: 본문 텍스트
- content.html: 본문 HTML
- meta.json: 원본 URL, 최종 URL, 저장 시간, 이미지 정보
- images 폴더: 본문 이미지


4. Edge 로그인 쿠키를 쓰고 싶을 때
----------------------------------

Chrome이 아니라 Edge에 네이버 로그인이 되어 있다면 아래 명령어로 실행합니다.

python save_naver_cafe_post.py --cookie-browser msedge


5. 직접 명령어로 실행
---------------------

설치:

pip install -r requirements.txt
python -m playwright install chromium

URL을 실행 중에 입력:

python save_naver_cafe_post.py

URL을 바로 지정:

python save_naver_cafe_post.py --url "https://cafe.naver.com/yourcafe/123456"

쿠키 가져오기를 끄고 실행:

python save_naver_cafe_post.py --no-cookie-import


6. 자동 저장이 실패할 때
------------------------

실행하기.bat에서 3번 "Manual HTML save"를 사용합니다.

이 방식은 평소 사용하는 브라우저에서 게시글을 열고, Ctrl+S로 저장한 HTML 파일을 도구가 다시 읽어서 정리합니다.


7. 문제 해결
------------

로그인이 필요하다는 메시지가 나오면 열린 브라우저에서 네이버 로그인 또는 카페 접근을 완료한 뒤 터미널에서 Enter를 누르세요.

게시글을 볼 수 없다는 메시지가 나오면 해당 계정이 카페에 가입되어 있는지, 게시판 열람 권한이 있는지 확인하세요.

추출에 실패하면 saved_posts/_debug 폴더를 확인하세요.


8. 주의 사항
------------

이 도구는 사용자가 열람 권한을 가진 게시글을 개인적으로 저장하기 위한 도구입니다.
저장한 콘텐츠의 이용과 재배포는 해당 카페와 게시글의 권리 및 정책을 따라야 합니다.

browser_profile 폴더에는 로컬 브라우저 세션 정보가 포함될 수 있습니다.
다른 사람에게 공유하거나 공개 저장소에 올리지 마세요.
