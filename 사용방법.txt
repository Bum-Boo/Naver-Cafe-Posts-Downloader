네이버 카페 아카이브 매니저 사용방법
====================================

이 앱은 네이버 카페 게시글을 내 컴퓨터에 저장하고, 저장된 글을 목록으로 관리합니다.


1. 설치
-------

처음 한 번만 실행합니다.

pip install -r requirements.txt
python -m playwright install chromium


2. 실행
-------

python app.py

또는 실행하기.bat를 열고 2번 "Open archive app"을 선택합니다.


3. 처음 사용
------------

1. 앱에서 네이버 로그인 세션 연결 버튼을 누릅니다.
2. 열린 Chromium 브라우저에서 네이버에 직접 로그인합니다.
3. 앱 상태 표시줄에 "로그인 세션이 저장되었습니다" 메시지가 뜰 때까지 기다립니다.
4. 앱으로 돌아와 네이버 카페 게시글 URL을 붙여넣습니다.
5. Download 버튼을 누릅니다.

로그인 세션은 data/browser_profile 폴더와 data/auth/naver_state.json에 저장됩니다.


4. 저장된 글 보기
-----------------

왼쪽 목록에서 저장된 게시글을 클릭하면 오른쪽에 메타데이터가 표시됩니다.

- Open Local Page: 저장된 view.html을 기본 브라우저에서 엽니다.
- Open Folder: 저장 폴더를 엽니다.
- Delete Archive: 확인 후 저장 폴더와 목록 항목을 삭제합니다.
- Refresh List: 저장 목록을 다시 불러옵니다.

상단의 세션 상태 표시에서 네이버 로그인 세션이 적용 중인지 확인할 수 있습니다.


5. 저장되는 파일
----------------

게시글마다 아래 파일이 저장됩니다.

- content.txt: 본문 텍스트
- content.html: 본문 HTML
- view.html: 오프라인으로 볼 수 있는 재구성 페이지
- meta.json: 원본 URL, 저장 시간, 이미지 정보
- images 폴더: 본문 이미지

저장 목록은 data/archive_index.json에 저장됩니다.


6. 문제 해결
------------

로그인 또는 접근 권한 메시지가 나오면 네이버 로그인 세션 연결을 먼저 진행하세요.

추출에 실패하면 saved_posts/_debug 폴더를 확인하세요.
